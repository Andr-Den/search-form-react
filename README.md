# search-form-react

## About project
The project implements a search by name from a API

## Getting Started

1. Clone this repository

```bash
git@github.com:Andr-Den/search-form-react.git
```

2. Install dependencies

```bash
npm install
```

3. Build project

```bash
npm run build
```

4. Run project

```bash
npm run start
```

## Used technologies

1. ReactJS
